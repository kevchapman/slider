Slider demos

This is a demonstration of the slider class

By default all sliders have touch controls for IOS and hardware accelerated css3 animation.

Hero Slider

This example shows a full width slider with an auto timer. The slider will auto rotate at a specified interval untill the user interacts with it. The timer will pause on mouseover and play on mouseleave.
